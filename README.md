# Weather-website
Flask based weather website. It diaplays weather using API and it's stored in database
### Required packages
#### Flask
`pip3 install Flask`
#### Flask-SQLAlchemy
`pip3 install flask-sqlalchemy`

### Useful documentations
#### API 
https://openweathermap.org/
#### Bulma CSS
